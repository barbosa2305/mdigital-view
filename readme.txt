1) Caso não tenha o NodeJS instalado, fazer a instalação. (https://nodejs.org/)

2) Para subir o servidor node, abrir o prompt de comando nesta pasta e executar os seguintes passos:
	2.1) Instalar as dependencias para rodar o servidor:
		npm install
		
	2.2) Para subir o servidor:
		nodemon server.js
		
	2.3) Para derrubar o servidor:
		Ctrl + C
		
	2.4) Entrar na pasta Public e seguir as intruções do arquivo Readme.txt contido nela.
